V1 CODEX TASK PROMPT

You are implementing V1 of a multi-platform customer service middle platform project named Omni-CSX.

You must strictly follow the repository rules defined in:
- DEVELOPMENT_RULES.txt
- PROJECT_CONSTRAINTS.txt

Your job is to generate and implement the V1 project skeleton and core functionality.

==================================================
1. V1 PRODUCT GOAL
==================================================

V1 only needs to deliver:
- multi-platform unified customer service intake
- unified order / shipment / after-sale context
- AI suggested reply with human confirmation before send

V1 does NOT include:
- recommendation engine
- customer profiling
- marketing tasks
- campaign system
- quality inspection center
- risk center
- ERP integration center
- VOC analysis
- training center
- Deep Agents
- automatic reply sending

==================================================
2. REQUIRED TECH STACK
==================================================

Backend:
- FastAPI
- PostgreSQL
- Redis

AI:
- LangChain
- LangGraph

Frontend:
- Next.js
- React

Use FastAPI for all business APIs, admin APIs, webhook APIs, OpenAPI output, and background task triggers.

Use LangChain only for:
- model abstraction
- tool calling
- retrieval integration
- structured output
- prompt composition
- suggestion generation

Use LangGraph only for:
- suggest_reply_graph

Do not use Deep Agents.

- load_context
