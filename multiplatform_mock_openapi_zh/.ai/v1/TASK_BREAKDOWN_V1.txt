This file assumes the existence of:
- DEVELOPMENT_RULES.txt
- PROJECT_CONSTRAINTS.txt
- CODEX_TASK_PROMPT.txt
- FIRST_EXECUTION_STEPS.txt

If there is any conflict, follow the rule files first.

SIMPLICITY RULE
==================================================

If you are uncertain:
- choose the simplest valid implementation
- keep architecture clean
- keep AI isolated
- keep provider logic isolated
- do not invent extra features
- do not anticipate V2 unless a placeholder is strictly necessary
